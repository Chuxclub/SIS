Rendez-vous dans le dossier : out/artifacts/SIS_jar dans un terminal de commande. 
Tapez ensuite : “java -jar SIS_jar.jar”.

Le jeu s’est à présent ouvert dans votre terminal de commande ! 
Vous pouvez taper “yes” pour charger une sauvegarde existante, 
ou appuyer sur Entrée pour lancer une nouvelle partie.
